# Pokemon API PHP - Clean Architecture

Proyecto backend en PHP puro estructurado con arquitectura por capas.

## Características

- Consumo de PokeAPI
- Persistencia en base de datos
- Envío de correos con PHPMailer
- Uso de variables de entorno (.env)

## Estructura

src/
  Controllers/
  Services/
  Repositories/
  Clients/
  Config/

## Instalación

composer install
cp .env.example .env
